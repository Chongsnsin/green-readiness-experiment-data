# Green-readiness Experiment Data

## Overview

This repository contains the de-identified participant-level data supporting Experiments 1 and 2 of the study:

**Matching low-carbon behavioural interventions to public green readiness**

The study examines how regulatory and affective interventions interact with individuals' green capability and green willingness to influence low-carbon behaviour.

## Files

- `data.xlsx`: de-identified participant-level data for Experiments 1 and 2

## Sample sizes

- Experiment 1: 4,069 valid participants
- Experiment 2: 6,075 valid participants

Only valid observations retained after applying the prespecified data-quality criteria are included in the public datasets.

## Experimental-condition coding

`Reg` and `Emo` are binary indicators identifying the regulatory and affective components of the experimental interventions.

### Regulatory intervention

- `Reg = 1`: high-regulation condition
- `Reg = 0`: low-regulation condition

### Affective intervention

- `Emo = 1`: high-affect condition
- `Emo = 0`: low-affect condition

The four experimental conditions are coded as follows:

- `Reg = 1, Emo = 1`: high-regulation, high-affect condition (`HRHA`)
- `Reg = 1, Emo = 0`: high-regulation, low-affect condition (`HRLA`)
- `Reg = 0, Emo = 1`: low-regulation, high-affect condition (`LRHA`)
- `Reg = 0, Emo = 0`: low-regulation, low-affect condition (`LRLA`)
